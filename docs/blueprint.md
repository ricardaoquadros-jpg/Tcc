# **App Name**: Code Clarity

## Core Features:

- Simple Login: User logs in by entering their name; no password or email required.
- Educational Content: Landing page features distinct educational sections focused on Frontend and Backend, highlighting key differences and primary technologies.
- Technology List: Display the main languages for front-end and back-end.
- Key Differences: Generative AI tool explains the core differences in Frontend vs. Backend development based on the user's learning goals.

## Style Guidelines:

- Primary color (Frontend): #4A90E2 (Technological Blue)
- Secondary color (Backend): #50E3C2 (Teal, modern)
- Background color: #F5F7FA (Very light grey)
- Text: #2C3E50 (Dark blue-grey)
- Highlight: #F39C12 (Vibrant yellow for buttons)
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text, chosen for its blend of modernity and readability, suitable for educational content.
- Simple, clear icons to represent frontend and backend technologies. Consider using icons that subtly suggest the function of each technology.
- Clean and intuitive layout, optimized for easy navigation and readability. Use distinct sections for frontend and backend explanations.
- Subtle transitions and animations to enhance user experience, such as a smooth page transition after login or animated highlights on key information.
